## 구현 완료

### 핵심 게임플레이
- 8방향 스프라이트 애니메이션 (idle/walk/attack × 8방향)
- 오프라인 모드 (Socket.IO 미연결 시 자동 전환)
- 영어 어휘 퀴즈 기반 전투 시스템 (공격 → 퀴즈 → 보상)
- 몬스터 AI (aggro/chase/attack 상태 머신)
- 몬스터 자동 리스폰

### 마을 안전 구역 시스템 (2026-03-15 추가)
- 서버(gameServer.ts): `isInsideMonsterSafeZone()` → 플레이어 안전구역 있으면 추격 중단
- 서버(monsterManager.ts): `isInSafeZone()` → `moveTowards()` 시 안전구역 진입 차단
- 클라이언트(WorldScene.ts): 오프라인 AI에서 `STARTER_TOWN_RECT` 체크 → 마을 추격 금지

### 전투 비주얼 (2026-03-15 추가)
- 피격 플래시 (HP 감소 시 스프라이트 알파 깜빡임 + 스파크 파티클)
- 사망 애니메이션 (scaleY 붕괴 + 페이드 아웃 + 폭발 파티클)
- 근거리 공격 강화 (큰 slash arc + 임팩트 플래시 타원)
- 원거리 공격 강화 (화살 궤적 + 명중 시 플래시 이펙트)

### HUD UI (2026-03-15 개편)
- 소울라이크 스타일 컴팩트 HUD
- 클래스별 컬러 초상화 원 (Guardian=파랑, Ranger=초록, Arcanist=보라, Sovereign=금)
- HP 그라디언트 바 (빨강, 12px 두께)
- MP 그라디언트 바 (파랑, 9px 두께)
- EXP 바 (금색, 5px 두께)
- 퀵슬롯 (1=HP포션, 2=MP포션, 3=이동주문서) - 키보드 1/2/3으로 즉시 사용
- 포션/주문서 재고 수량 표시
- 버프 아이콘 표시

### 인벤토리/장비
